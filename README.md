
#  MDCP Song Picker

一款优雅的桌面应用，旨在帮助音游玩家管理他们的MDCP个人歌曲收藏，并从中随机抽取歌曲进行练习或娱乐。告别“选择困难症”，让每一次游玩都充满惊喜！

> **软件内置MDCP部分曲库**

##  下载
一般而言，你可以从 [GitHub Release](https://github.com/saba-futai/mdcp/releases) 下载最新适用于Windows的版本。


##  主要功能

- **歌曲管理**: 轻松添加、编辑和删除你的歌曲信息，包括歌名、曲包/作者、难度等级和自定义卡片颜色。
- **收藏系统**: 一键标记你喜爱的歌曲，并可以随时筛选只显示收藏列表。
- **强大的筛选与排序**:
    - **实时搜索**: 按歌名快速找到你想要的歌曲。
    - **多种排序**: 支持按歌名 (A-Z, Z-A) 和难度 (正序, 逆序) 进行排序。
- **智能随机抽取**:
    - **自定义数量**: 自由选择一次抽取多首歌曲。
    - **范围筛选**: 指定抽取的难度等级范围 (例如 Lv. 9.5 - 11.0)。
    - **从搜索结果抽取**：可以从搜索结果中随机抽取歌曲。
    - **收藏优先**: 可以设置仅从你的收藏列表中进行抽取。
- **现代化界面**: 采用美观的深色主题和流畅的动画效果，提供卓越的用户体验。
- **性能优化**: 使用虚拟滚动（懒加载）技术，即使拥有数千首歌曲也能保持流畅运行。
- **跨平台**: 基于 Wails 构建，可轻松打包为 Windows, macOS 和 Linux 的原生桌面应用。

##  技术栈

- **后端**: Go
- **前端**: Vue.js 3 (Composition API)
- **框架**: [Wails v2](https://wails.io/)
- **样式**: Modern CSS with Flexbox & Grid

##  快速开始

请确保你的开发环境已经准备就绪。

### 1. 环境准备

你需要安装以下软件：

- **Go**: 1.18 或更高版本。 [安装指南](https://go.dev/doc/install)
- **Node.js**: 16 或更高版本 (包含 npm)。 [安装指南](https://nodejs.org/en/)
- **Wails CLI**: Wails 的命令行工具。
  ```shell
  go install github.com/wailsapp/wails/v2/cmd/wails@latest
  ```

你可以运行 `wails doctor` 来检查环境是否配置正确。

### 2. 安装与运行

1.  **克隆仓库**
    ```shell
    git clone https://github.com/your-username/rhythm-game-picker.git
    cd rhythm-game-picker
    ```

2.  **安装前端依赖**
    ```shell
    # 进入 frontend 目录
    cd frontend

    # 安装依赖
    npm install
    ```

3.  **开发模式运行**
    回到项目根目录，运行以下命令启动应用：
    ```shell
    # 这会启动一个带热重载功能的应用，方便开发
    wails dev
    ```

### 3. 构建应用

当你完成开发，想要构建一个可独立分发的文件时，运行：

```shell
# 这会在 build/bin 目录下生成对应你操作系统的可执行文件
wails build
```

##  使用指南

1.  **添加歌曲**: 点击右上角的 `➕ 添加歌曲` 按钮，在弹出的模态框中填写歌曲信息，然后点击 `保存`。
2.  **查找歌曲**:
    - 在顶部的**搜索框**中输入歌名进行实时搜索。
    - 使用**排序下拉框**来改变歌曲的排列顺序。
    - 勾选 `只显示收藏` 来快速查看你喜欢的歌曲。
3.  **收藏/编辑/删除**:
    - 将鼠标悬停在歌曲卡片上。
    - 点击卡片右上角的 `☆` 星星图标来收藏或取消收藏。
    - 点击卡片右下角的 `✏️` 和 `🗑️` 图标来编辑或删除歌曲。
4.  **随机抽取**:
    - 点击右上角的 `🎲 随机抽取` 按钮。
    - 在设置面板中，选择你想要的**抽取数量**、**等级范围**，以及是否**仅从收藏中抽取**。
    - 点击 `开始抽取`，即可看到为你精心挑选的结果！
